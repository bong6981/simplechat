# simplechat

웹소켓을 학습하기 위해 간단하게 채팅 프로그램을 구현해보았습니다. 
- 브랜치 `basic-websocket` : sockJS를 사용하지 않고 구현했습니다. 
- 브랜치 `sockJS-websocket` :  프론트를 구현하지 않고 기능만 구현했습니다. 
- 브랜치 `main` : sockJS와 프론트를 구현한 최종본 입니다.
